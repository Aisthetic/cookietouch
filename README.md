###### DofusTouch Bot with Electron React/TypeScript.

You need **Yarn** in order to use this bot properly.
```sh
npm install -g yarn
```

## Dev
```sh
yarn dev
```

## Lint
```sh
yarn lint
```

## Lint Fix
```sh
yarn lint:fix
```
