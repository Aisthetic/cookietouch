You need **Yarn** in order to use this bot properly.

[Yarn Install](https://yarnpkg.com/lang/en/docs/install/)

## Installation

```bash
git clone https://github.com/yovanoc/cookietouch/
cd cookietouch && yarn
```

## Dev

```bash
yarn dev
```

## Lint

```bash
yarn lint
```

## Lint Fix

```bash
yarn lint:fix
```
